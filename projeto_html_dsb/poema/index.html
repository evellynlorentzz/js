O MUNDO DO HTML


Com tags e códigos, um mundo a criar,

HTML, a base, para a web começar.

De títulos a parágrafos, tudo a organizar,

Um quebra-cabeça digital, para explorar.


Com imagens e links, a página a colorir,

Um universo virtual, para sonhar e sentir.

div e span, para estruturar e definir,

HTML, a linguagem, para a web definir.


Do simples ao complexo, um código a entender,

Com CSS, a beleza, para o mundo ver.

JavaScript, a interação, para o usuário encantar,

HTML, o alicerce, para a web levantar.

- Evellyn Lorentz
